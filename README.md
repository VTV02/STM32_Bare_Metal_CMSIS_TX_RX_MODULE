# STM32_Bare_Metal_CMSIS_TX_RX_MODULE using STM32F411RE NUCLEO BOARD 
Using CMSIS driver of ST to develop program UART
Using Hercules send '1' to MCU when MCU receives '1', turn on LED PA5.
                    '0'                               turn off LED PA5
